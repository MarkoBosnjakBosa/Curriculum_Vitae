Curriculum Vitae

Features:  
REST API by using Express.js  
Login system by using Express-session and MemoryStore  
Password encryption by using Bcryptjs  
File upload by using Formidable  
Multilingual pages (English and German)  
Frontend customizations by using Bootstrap, jQuery and Font Awesome icons  
RememberMe by using the localStorage API  
Frontend custom form validations  
Backend custom form validations (middleware)  
Sending emails by using Nodemailer  
MongoDB and Mongoose (queries, limits, sorts, pages)
